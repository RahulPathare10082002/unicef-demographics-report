# UNICEF Global Demographic Dynamics Report

### [🔗 View Live Report](https://RahulPathare10082002.github.io/unicef-demographics-report/)

## Project Overview
This project provides a data-driven analysis of global demographic shifts, focusing on crude birth rates and child population distribution from 1960 to the present. The report was generated using **Quarto** and **Python** as part of a data visualization and demographics study.

## Key Features
* **Trend Analysis:** Visualizing the decline in global average birth rates over the last 60 years.
* **Geospatial Mapping:** A country-wise distribution of child populations.
* **Reproducible Workflow:** Developed in Google Colab with data sourced from reliable global health databases.

## Tech Stack
* **Language:** Python
* **Tools:** Quarto, Google Colab, GitHub Pages
* **Libraries:** `pandas`, `plotnine`, `geopandas`, `polars`
